JPEG-LS Decoding with WebAssembly

Select an image or drag and drop a JPEG-LS file. Files dropped here remain local in your browser, they are not uploaded anywhere. Note that images with bitsPerSample > 8 are read as 16 bit signed integers so we can display our DICOM sample images properly. Images that are 16 bit unsigned will not be displayed properly!
